We're working hard on creating a user friendly GUI submission tool for AcousticBrainz. Sadly, it isn't ready just yet -- we hope to release it very soon.

Do it yourself install

If you prefer to build your own install of this software, you'll need Python (version 2.6 or later), along with the requests library. Follow the instructions in acousticbrainz-client.

Part of the source installation requires a static extractor binary. We have these versions for Linux, Mac & Windows available:

  • linux i386 extractor static binary (4.9Mb)
  • linux x86_64 extractor static binary (5.6Mb)
  • mac 32 bit extractor static binary (4.62Mb)
  • mac 64 bit extractor static binary (4.62Mb)
  • win 64 bit extractor static binary (6.72Mb)
  • win 32 bit extractor static binary (6.32Mb)
