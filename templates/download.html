Do it yourself install

If you prefer to build your own install of this software, you'll need a recent Linux install. Follow the instructions in acousticbrainz-client.

Part of the source installation requires a static extractor binary. We have these versions for linux & mac available:

  • linux i386 extractor static binary (4.9Mb)
  • linux x86_64 extractor static binary (5.6Mb)
  • mac 32 bit extractor static binary (4.62Mb)
  • mac 64 bit extractor static binary (4.62Mb)
